main.html : contient le squelette HTML de la page.

web. py : script python qui lance un serveur local pour les tests.

style.css : feuille de style du projet.

app.js : effectue simplement la création de l'application AngularJS

global_map.js : contient les fonctions relatives aux actions effectuées sur la carte.

controllers.js : contient les contrôleurs affectés aux différentes sections (Liste des arbres, Liste des doublons, Modification des coordonnées)

service.js : contient le service API permettant la connexion à la base de données centrale des arbres hébergée sur "http://rencontres-arbres.herokuappcom.api"

dups.json : contient les doublons des arbres sous la forme d'une liste d'objets JSON.
